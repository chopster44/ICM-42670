# ICM-42670
A driver for the TDK InvenSense ICM-42670 6-axis motion sensor. 

The only dependencies of this driver is `Wire.h` and `Arduino.h`.
## Instalation

To install add:
`https://github.com/chopster44/ICM-42670.git` to your `platformio.ini` file.
